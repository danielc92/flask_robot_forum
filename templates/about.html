About

Robot forum is a forum-based web application built by Daniel Corcoran. The purpose of creating this application was to learn about the technical aspects of a relational database talking to a web application, feeding complex joins into rendering templates and deploying a web application to a PAAS (platform as a service). Themes such as pagination, ORMs, querying, rendering templates, dynamic routing, responsive design are covered here.

Details

  • Gunicorn used as an WSGI
  • python 3.6.5 was used as a backend
  • Jinja2 as a template engine
  • flask used as a micro web framework
  • flask-sqlalchemy used as an ORM tool to map the database to the app
  • sqlite3 used to store the data
  • Web application is currently being hosted on heroku, a platform as a service provider.

Features

For source information please visit this link

  • Side panel with new members, starred threads and recent comments
  • Members route listing all registered members
  • Profile route listing details about a member, including posts and stars accumulated
  • Dynamic template design
  • News route
  • About, sources, features routes
  • Search functionality and pagination functionality built into members and threads routes
  • Robot avatars and thread text were generated using a third part service
